# Egyszerű Konzolos Dáma (Checkers) – 2 játékos

Ez egy klasszikus **dáma** (angolul: checkers) játék Pythonban, amit két ember játszhat egymás ellen a terminálban / parancssorban.

## Jellemzők

- 8×8-as klasszikus tábla
- Fekete (b / B) felülről indul, Fehér (w / W) alulról
- Normál bábuk csak előre lépnek, királyok mindkét irányba
- **Kötelező az ütés**, ha van lehetőség rá
- Többszörös ütés támogatott (egy lépésben több bábu is levehető)
- Királysorra érés után a bábu királlyá válik
- Győzelem: amikor az ellenfélnek elfogynak a bábui

## Telepítés és futtatás

Nincs szükség semmilyen külső könyvtárra – tiszta Python 3.
